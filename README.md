# Divorce-lawyers-LLM
이혼판례 판단 LLM

### create conda env

```
conda env create -f llm.yaml
```

### Run Fastapi backend

```
uvicorn main:app --reload
```

### Run Streamlit Frontend

```
streamlit run app.py
```

### Access Streamlit app
http://localhost:8501
